Password Security

Build passwords that are hard to crack.

Use the checker below to test password strength in your browser, understand what weakens a password, and learn safer habits for every account you use.

Risk

Weak passwords spread damage fast

If one reused password is exposed in a breach, attackers often try it immediately on your email, banking, shopping, and social accounts.

Habit

Unique beats clever

A unique passphrase for each account is more valuable than a single "smart" password reused everywhere.

Layer

Two-factor still matters

Even a strong password is safer when paired with two-factor authentication and a trusted password manager.

What works

Good password habits

  • Use a different password for every important account
  • Prefer long passphrases over short complicated strings
  • Turn on two-factor authentication wherever possible
  • Store passwords in a trusted password manager
Avoid this

Common mistakes

  • Reusing the same password on email, banking, and social media
  • Using names, birthdays, or phone numbers that others can guess
  • Using obvious patterns like 123456, qwerty, or password123
  • Saving passwords in plain text notes or screenshots
Quick checklist

Before you trust a password

  • Is it at least 12 characters long?
  • Is it unique to this account?
  • Would it still be hard to guess if someone knows your birthday or name?
  • Have you enabled two-factor authentication as a backup layer?
STEP 01

Start with length

Longer passwords are usually stronger than shorter ones. A passphrase with 4 random words is easier to remember and much harder to brute-force.

STEP 02

Make every account unique

If one reused password is leaked in a breach, attackers immediately try it on your email, banking, and social apps.

STEP 03

Protect the password itself

Even strong passwords are safer with two-factor authentication and a password manager that stores them securely.
